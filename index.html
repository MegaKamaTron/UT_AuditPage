Help & Features

everything the tool can do

🏠 Home & Stories

Multiple Stories▶
Click the ▾ arrow next to the story title in the masthead to switch between projects or create a new one. Each story has its own chapters, drafts, edits, and images. Your profile and reader list are shared across all stories.
Cover Image▶
Click the image zone at the top of the Home page to upload a cover (recommended 1000×600 px). Hover to change or remove it.
Writing Goal▶
Set a target word count and deadline. The progress bar updates as you write across all chapters.

✍️ Draft

Floating Toolbar▶
The formatting toolbar floats as a pill at the bottom of the screen while you're on the Draft tab. It contains all formatting options (bold, italic, headings, lists, alignment, font, size), plus Save and Focus buttons. It disappears when you leave the Draft tab.
Focus Mode▶
Click ⛶ Focus for fullscreen, distraction-free writing. Press Esc or use the exit bar at the bottom to return.
Font & Size in Toolbar▶
The floating toolbar includes a font family picker (Serif, Sans-serif, Dyslexic) and a font size picker. These apply per-selection, not globally. Use Settings to change the global app font.
Save Draft▶
Drafts auto-save every 800ms. The Save Draft button forces an immediate save with visual confirmation. Cmd+S / Ctrl+S also saves.

👁 Beta Mode & Readers

Beta Mode (Writer)▶
Click ⬡ Beta Mode in the masthead to enter a read-only preview of your draft. The floating bar lets you log edits and set bookmarks — useful for self-editing passes.
Beta Reader Access▶
Go to the Readers tab and add a reader's email. Share the file with them and have them click Open as Reader and enter their email. They get access to Draft and Edit Suite only — they can highlight passages, log edits, and set bookmarks. All writer-only controls disappear.
Starring Readers▶
Click the ☆ star button on any reader row to mark them as a favourite. Starred readers sort to the top of the list.

💾 Export & Data

Export to PDF▶
Use the ↓ Export dropdown and choose PDF. This opens a print-formatted view of the current chapter draft using your browser's built-in PDF export. For best results, choose "Save as PDF" in the print dialog.
JSON Backup▶
Export all stories, chapters, and profile data as a single JSON file. Import it on any device to restore everything. Keep regular backups — localStorage can be cleared by the browser.
